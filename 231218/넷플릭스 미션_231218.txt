<미션>

1.각 영화 카드 아이템 클릭 시, 영화 상세페이지로 이동!!!
- 각 영화는 고유의 ID값 보유
- 해당 ID값을 활용해서 영화 상세페이지로 이동
- 상세페이지 > 영화 개봉년도 / 리뷰작성자수 / 영화장르 등

2.Home 컴포넌트 우측 상단, 검색바 기능 구현
- 검색바에 입력되는 값을 쿼리 전달 => 쿼리값을 찾아서 결과물출력
- filter 함수를 활용해서 값을 검색

3.Home 컴포넌트 좌측 상단, Movies를 클릭했을 때
- 현재 상영중인 영화를 한 페이지 혹은 여러 페이지 형태로 출력
